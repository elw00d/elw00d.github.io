Getting started

  • Download the latest release
  • Open ConsoleFramework.sln solution in Visual Studio or MonoDevelop (aka Xamarin Studio).
  • If you are working in Linux or Mac, select DebugMono configuration instead of default Debug
  • Set Examples as startup project
  • In Examples project settings choose one of entry points
  • Build & run it!

Participating

If you have found a bug, please, submit issue about it on github. If you want to propose some missing functionality, you can create the issue too.
